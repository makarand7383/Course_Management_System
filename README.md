# 🚀 Course Management System

A full-stack **Course Management System** with role-based authentication, built using React and Spring Boot.

---

## 📌 Features

### 👨‍💼 Admin
- Add, update, and manage courses
- View and manage students
- Track enrollments
- Enroll students into courses

### 👨‍🎓 Student
- Browse available courses
- Request course enrollment
- View personalized dashboard

---

## 🛠️ Tech Stack

**Frontend**
- React
- CSS

**Backend**
- Spring Boot
- REST APIs

**Database**
- SQL

---
## 📂 Project Structure

Course_Management_System/
├── frontend/ # React App
├── backend/ # Spring Boot API
---

## ⚙️ Setup Instructions

### 🔹 Frontend
```bash
cd frontend
npm install
npm start
```
🔹 Backend
```bash
cd backend
mvn spring-boot:run
```
